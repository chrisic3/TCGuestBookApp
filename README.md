# TCGuestBookApp

Build a Console Guest Book. Ask the user for their name and how many are in 
their party. Keep track of how many people are at the party. At the end, 
print out the guest list and the total number of guests.

He uses a List for the names and a single total variable for the guest count. 
I chose a Dictionary so that the count's are paired with the guests in case
that is ever wanted.
