Build a Console Guest Book. Ask the user for their name and how many are in 
their party. Keep track of how many people are at the party. At the end, 
print out the guest list and the total number of guests.

He uses a List for the names and a single total variable for the guest count. 
I chose a Dictionary so that the count's are paired with the guests in case
that is ever wanted.

Program.cs
Welcome User
Dictionary<string, int> = Party.GetPartyInfo()
Party.PrintGuestBookInfo(Dictionary<string, int>)
Say Goodbye

Party.cs
GetPartyInfo()
{
	Ask for party name, ask for party count, store in dictionary. 
	Ask for more guests.
	Return Dictionary.
}
PrintGuestBookInfo(Dictionary<string, int>)
{
	Loop through and print each party name.
	Sum and print the total number of guests.
}
